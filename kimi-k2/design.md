# AI Expert CV Website - Design Style Guide

## Design Philosophy

### Visual Language
**Modern Intelligence**: Clean, sophisticated design that reflects cutting-edge AI expertise while maintaining human approachability. The aesthetic balances technical precision with creative innovation, positioning the expert as both knowledgeable and accessible.

### Color Palette
**Primary Colors**:
- Deep Navy (#1a1d29) - Professional authority and trust
- Electric Blue (#00d4ff) - Innovation and technology
- Soft White (#fafafa) - Clarity and openness

**Accent Colors**:
- Warm Gray (#6b7280) - Supporting text and subtle elements
- Success Green (#10b981) - Positive feedback and achievements
- Warning Amber (#f59e0b) - Attention and call-to-actions

### Typography
**Display Font**: Inter Bold - Modern, technical, highly legible
**Body Font**: Inter Regular - Clean, professional, excellent readability
**Code Font**: JetBrains Mono - For technical demonstrations

### Visual Hierarchy
- Large, bold headings that command attention
- Generous white space for clarity and focus
- Strategic use of color to guide user attention
- Consistent spacing using 8px grid system

## Visual Effects & Animation

### Core Libraries Used
1. **Anime.js** - Smooth micro-interactions and element animations
2. **ECharts.js** - Data visualization for skills and project metrics
3. **p5.js** - Creative coding background effects and AI visualizations
4. **Pixi.js** - High-performance visual effects for hero section
5. **Splitting.js** - Advanced text animations and reveals
6. **Typed.js** - Typewriter effects for dynamic content
7. **Splide.js** - Smooth carousels for testimonials and projects
8. **Matter.js** - Physics-based interactions for engaging elements

### Animation Strategy
**Subtle & Purposeful**: All animations serve to enhance user understanding and engagement without being distracting.

**Performance-First**: Optimized animations that work smoothly across devices while maintaining visual impact.

### Header Effects
**AI Neural Network Visualization**: Using p5.js to create a dynamic, animated neural network background that responds to user interaction. The visualization uses connected nodes with flowing data particles to represent AI processes in an artistic, non-technical way.

**Typewriter Hero Text**: Large, bold typography with typewriter animation revealing key value propositions about AI expertise and coaching services.

### Scroll Motion Effects
**Progressive Disclosure**: Content sections fade in and slide up as users scroll, creating a sense of discovery and progression through the expert's capabilities.

**Parallax Elements**: Subtle background elements move at different speeds to create depth and visual interest without overwhelming the content.

**Skill Meter Animations**: Interactive skill assessments and experience metrics animate into view with satisfying progress bar fills and counter increments.

### Hover Effects
**3D Card Lifts**: Project cards and service offerings lift with shadow expansion and subtle rotation on hover, creating tactile feedback.

**Color Morphing**: Buttons and interactive elements shift colors smoothly, with electric blue highlights appearing on hover to reinforce the tech theme.

**Content Previews**: Hovering over project thumbnails reveals overlay information with smooth fade transitions and scale effects.

### Background Design
**Consistent Theme**: Dark navy background maintained throughout all sections with subtle texture overlay for depth.

**Decorative Elements**: Geometric AI-inspired patterns in the background using low-opacity electric blue accents that don't interfere with content readability.

**Section Differentiation**: Instead of background color changes, sections are differentiated through content layout, spacing, and strategic use of accent colors.

## Mobile Responsiveness
**Mobile-First Approach**: All animations and effects scale appropriately for mobile devices, with simplified interactions for touch interfaces.

**Performance Optimization**: Reduced animation complexity on mobile to maintain smooth performance while preserving visual impact.

## Accessibility Considerations
**Reduced Motion Support**: All animations respect user preferences for reduced motion.

**High Contrast**: Color combinations meet WCAG AA standards for accessibility.

**Keyboard Navigation**: All interactive elements are accessible via keyboard navigation.

## Brand Consistency
**Professional Authority**: Design elements reinforce expertise and credibility in AI field.

**Approachable Innovation**: Balance between cutting-edge technology and human-centered design.

**Trust Building**: Visual consistency and quality execution build confidence in professional capabilities.
